Penjelasan table

Kita harus menambahkan border, kalo tidak menambahkan border di table maka tidak akan muncul kotak garis.
cellspacing adalah antar kotak dengan kotak yang lainnya, kalo tidak mau ada antar kotak tulis saja (cellspacing="0").
Selanjutnya cellpadding yaitu, memberikan ruang/jarak teks di dalam table agar tidak terlalu mepet.
